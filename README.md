## سایت فروشگاهی
در این پروژه کار با فایل های view , models, admin, urls, و... تمرین شده است. همینطور بخش ثبت نام و ورود کاربر پیاده سازی شده است، کار با ajax پیاده شده. به طور کلی بخش هایی که پیاده سازی شده است:
۱.ثبت نام کاربر
۲.ورود به حساب 
۳.درگاه پرداخت
۴.مرتب شدن محصولات 
۵.پنل ادمین 
۶.تماس باما
۷.درباره ما
